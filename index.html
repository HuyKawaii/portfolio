Link bomb

A fun little 2D game. Where the player place bomb on the screen, close by bombs are linked together by fuse, when one of them detonate ,it burns all fuses connected to it and detonate other bombs. Player can only detonates bomb at their base(on the white area) and enemies fall from the top and damage player base when they touch it. Here I learn more about game saving, and post processing.

Jeopardy maker

Simulate the American TV show with Jeopardy, allow a game master to create lobby and let friends join online. I make this project to learn more about UI elements in Unity and Netcode for GameObject to implement multiplayer games.

Outside

A 3D multiplayer survival game with procedure map generation (each playthrough is a different map) including terrain and resource generation, simple combat system, player animation, inventory system (with drag and drop UI), equipment system with a stat system, crafting system, day-night cycle. The game use Netcode for GameObject of Unity to implement relay connection between players.
